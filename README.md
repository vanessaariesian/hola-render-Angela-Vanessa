# Hola Render - Static Web Deployment

Proyecto de despliegue de una página web estáica utilizando GitHub y Render con entornos de Preproducción y Producción.

## Overview

Esta aplicación consiste en una página HTML sencilla desplegada automáticamente desde GitHub mediante [Render](https://render.com?utm_source=chatgpt.com).

El proyecto utiliza dos entornos independientes:

- **Preproduction** → rama `pre`
- **Production** → rama `main`

## Tech Stack

- HTML5
- Git
- GitHub
- Render
- Python
- Pytest
- Requests
